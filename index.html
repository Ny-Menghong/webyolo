  • бһҷбһӣбҹӢбһ–бһёбһҖбҹ’бһ”бһҪбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ YOLO бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—

бһ…бһјбһӣбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһ“бһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһ‘бһ¶бһҸбҹӢбһҮбҹҶбһ“бһ¶бһүбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһңбһҸбҹ’бһҗбһ»бҹ” бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбҹҖбһ“бһҠбҹ„бһҷбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбҹ”

бһҳбһҫбһӣ Demo

бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ вҖў бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҸбһҘбһҺбһ‘бһ¶бһ“бһ‘бҹҒ

бһҳбҹҒбһҡбҹҖбһ“бһ‘бһёбҹЎ

бһҸбһҫбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (Artificial Intelligence - AI) бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?

бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (Artificial Intelligence бһ¬ AI) бһӮбһәбһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ бһҡбҹҖбһ“бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһңбһ·бһ—бһ¶бһӮбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ”бһ¶бһ“бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” AI бһўбһ¶бһ…бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһҹбҹ’бһҸбһ¶бһ”бҹӢбһҹбҹҶбһЎбҹҒбһ„ бһ”бһҖбһ”бҹ’бһҡбҹӮбһ—бһ¶бһҹбһ¶ бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ“бҹ” бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ AI бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҹбһ»бһҒбһ¶бһ—бһ·бһ”бһ¶бһӣ бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶ бһ’бһ“бһ¶бһӮбһ¶бһҡ бһҖбһҹбһ·бһҖбһҳбҹ’бһҳ бһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI

AI бһўбһ¶бһ…бһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһӣбҹҶбһ“бһ¶бҹҶ бһ“бһ·бһ„бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ”бһ‘бһӣбҹ’бһҳбһҫбһҹбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһўбһ¶бһ…бһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһ•бһӣбһ·бһҸбһ•бһӣбһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбҹ” - Natural Language Processing (NLP) бһ“бһ·бһ„ Computer Vision бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

бһҹбһҡбһ»бһ”

бһ“бҹ…бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹҒбҹҮ AI бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӣбҹҶбһ“бһ¶бҹҶ (patterns) бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһңбһ¶бһ”бҹ’бһҡбһҫ algorithm бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҗбһ¶ бһҸбһҫбһўбҹ’бһңбһёбһҮбһ¶бһўбҹ’бһңбһёбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ AI бһўбһ¶бһ…бһҡбҹҖбһ“бһҗбһ¶ бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бҹ’бһ“бҹӮбһҖ бһ…бҹ’бһҡбһҳбһ»бҹҮ бһ“бһ·бһ„бһҳбһ¶бһҸбҹӢ бһўбһ¶бһ…бһҮбһ¶бһҳбһ»бһҒбһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҠбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸ вҖңбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹҶбһ”бһјбһ„вҖқ бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”

3. вҡҷпёҸ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ (Model Training)

бһҳбҹүбһјбһҠбҹӮбһӣ (Model) бһӮбһәбһҮбһ¶бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ¬ algorithm бһҠбҹӮбһӣ AI бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбҹ” бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹҒбҹҮ AI бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҠбҹ„бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһҠбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹ (error) бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҡбһ бһјбһҸбһҠбһӣбҹӢбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбҹ’бһўбҹ”

4. рҹҺҜ бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– (Decision Making & Action)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ AI бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ AI бһўбһ¶бһ…бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҗбһ¶бһҮбһ¶бһҶбҹ’бһҳбһ¶ бһ¬бһҶбҹ’бһҖбҹӮ, бһ¬бһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ’бһ“бһ¶бһӮбһ¶бһҡбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ¬бһўбһҸбҹӢбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһ·бһҸ AI бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҠбһјбһ…бһҮбһ¶ бһ”бһҫбһҖбһ”бһҡбһҡбһҗбһҷбһ“бҹ’бһҸ бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ бһ¬бһҺбҹӮбһ“бһ¶бҹҶбһ•бһӣбһ·бһҸбһ•бһӣбҹ”

5. рҹ”„ бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ (Continuous Improvement)

AI бһҳбһ·бһ“бһҲбһ”бҹӢбһҡбҹҖбһ“бһ‘бҹҒбҹ” бһ“бҹ…бһ–бҹҒбһӣбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ бһңбһ¶бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ— бһ“бһ·бһ„ feedback бһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ—бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһ‘бҹ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһңбһ·бһү бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҶбҹ’бһӣбһ¶бһҸ бһ“бһ·бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһ„бһҳбһ»бһ“бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ вҖңFeedback LoopвҖқ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷ AI бһҖбҹӮбһӣбһҳбҹ’бһўбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

бһҳбҹҒбһҡбҹҖбһ“бһ‘бһёбҹӨ

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹғбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (Types of AI)

бһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (AI) бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹӮбһ„бһ…бҹӮбһҖбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹғбһ—бһ¶бһ–бһҶбҹ’бһӣбһ¶бһҸ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһ·бһҸбҹ” бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһңбһ·бһңбһҸбҹ’бһҸбһ“бҹҚбһ–бһё AI бһҹбһ¶бһҳбһүбҹ’бһүбһ‘бҹ… AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ”

1. рҹӨ– Narrow AI (AI бһҸбһјбһ… / AI бһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸ)

Narrow AI бһӮбһәбһҮбһ¶ AI бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһӮбһ·бһҸбһҖбҹ’бһҡбҹ…бһ–бһёбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– Face Recognition, Google Translate, Chatbot бһ“бһ·бһ„ YOLO Object Detectionбҹ” AI бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ‘бһјбһ‘бҹ…бһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”

2. рҹ§  General AI (AI бһ‘бһјбһ‘бҹ…)

General AI бһӮбһәбһҮбһ¶ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ·бһҸ бһ“бһ·бһ„бһҡбҹҖбһ“бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘ бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹӮбһҳбһҪбһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹҒбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһңбһ¶бһўбһ¶бһ…бһҡбҹҖбһ“бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ бһ“бһ·бһ„бһҷбһӣбҹӢбһ–бһёбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһ“бһ»бһҹбҹ’бһҹбһ”бһ¶бһ“бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ General AI бһ“бҹ…бһҸбҹӮбһҮбһ¶бһӮбҹҶбһ“бһ·бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

LEARNING FROM DATA

Machine Learning

Machine Learning gives computers the ability to improve at a task by learning from examples, instead of following hand-written rules.

Supervised Learning

Learns from labeled examples вҖ” inputs paired with correct answers вҖ” to predict labels for new data.

e.g. spam detection, YOLO training

Unsupervised Learning

Finds hidden structure in unlabeled data by grouping similar examples together.

e.g. customer segmentation

Reinforcement Learning

An agent learns by trial and error, receiving rewards or penalties for its actions.

e.g. game-playing agents, robotics
NEURAL NETWORKS

Deep Learning

Deep Learning uses layered neural networks to automatically learn features directly from raw data like pixels.

Neural Networks

Layers of connected nodes ("neurons") that transform input signals into predictions.

Hidden Layers

Intermediate layers between input and output that build increasingly abstract features.

Weights & Bias

Weights scale each input's importance; bias shifts the result вҖ” both are learned during training.

Activation Functions

Non-linear functions (like ReLU) that let networks model complex relationships.

interactive signal propagation

YOLO (You Only Look Once)

YOLO бһӮбһәбһҮбһ¶ Model бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһңбһҸбҹ’бһҗбһ» (Object Detection) бһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһҫбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҸбҹӮбһҳбһҪбһҷбһҠбһ„бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ бһҫбһҷбһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ‘бһёбһҸбһ¶бҹҶбһ„ (Location) бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһңбһҸбҹ’бһҗбһ» (Class) бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸ (Real-time) бһҠбһјбһ…бһҮбһ¶ Camera, Self-driving car бһ“бһ·бһ„ AI Surveillanceбҹ”

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё YOLO бһӣбһҝбһ“?

бһҳбһ»бһ“бһ“бҹҒбҹҮ Object Detection бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“ (Multi-stage) бһҠбһјбһ…бһҮбһ¶бҹ– бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Region бһҮбһ¶бһҳбһ»бһ“ вҶ’ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһңбһҸбҹ’бһҗбһ»бһҖбҹ’бһ“бһ»бһ„ Region бһ“бҹ„бҹҮбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ YOLO бһ”бһҳбҹ’бһӣбҹӮбһ„бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһ‘бҹ…бһҮбһ¶ бһ”бһүбҹ’бһ бһ¶ Regression бһҸбҹӮбһҳбһҪбһҷбһҠбһ„ бһҠбҹ„бһҷбһ‘бһ¶бһҷ bounding boxes бһ“бһ·бһ„ class бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҸбҹӮбһҳбһҪбһҷ (single forward pass)бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһӣбһҝбһ“бһҮбһ¶бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бһ¶бһҹбҹӢбҹ—бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ”

One-Stage Detector (бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһҪбһҷбһҮбҹҶбһ бһ¶бһ“)

YOLO бһ”бҹӮбһ„бһ…бҹӮбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһҮбһ¶ Grid (бһҸбһ¶бһҡбһ¶бһ„бһҸбһјбһ…бҹ—)бҹ” Grid бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бһҷбһҗбһ¶бҹ– бһҸбһҫбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ¬бһўбһҸбҹӢ? бһ”бһҫбһҳбһ¶бһ“ бһңбһ¶бһ“бһ№бһ„бһ‘бһ¶бһҷ bounding box бһ“бһ·бһ„ class probability бһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ” бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Region Proposal бһҠбһјбһ… R-CNN бһ‘бҹҒбҹ”

Bounding Box бһ“бһ·бһ„ Confidence Score

Bounding Box бһӮбһәбһҮбһ¶бһҹбҹҠбһ»бһҳ (Rectangle) бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһёбһҸбһ¶бҹҶбһ„бһңбһҸбҹ’бһҗбһ»бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһңбһ¶бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҠбһјбһ…бһҮбһ¶бҹ– x, y (бһ‘бһёбһҸбһ¶бҹҶбһ„) бһ“бһ·бһ„ width, height (бһ‘бҹҶбһ бҹҶ)бҹ” YOLO бһҖбҹҸбһ•бҹ’бһҸбһӣбҹӢ Confidence Score бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҡбһ·бһҸбһҮбһҝбһҮбһ¶бһҖбҹӢбһҗбһ¶ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ box бһ“бҹ„бҹҮбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

Class Prediction бһ“бһ·бһ„ NMS

бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷ Bounding Box YOLO бһҖбҹҸбһ‘бһ¶бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһңбһҸбҹ’бһҗбһ» (Class Probability) бһҠбһјбһ…бһҮбһ¶ бһҳбһ“бһ»бһҹбҹ’бһҹ, бһЎбһ¶бһ“, бһҶбҹ’бһҖбҹӮбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бҹ’бһҡбһҫ Non-Maximum Suppression (NMS) бһҠбһҫбһҳбҹ’бһ”бһёбһӣбһ»бһ” box бһҠбҹӮбһӣбһҹбҹ’бһ‘бһҪбһ“бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҸбҹӮ box бһҠбҹӮбһӣбһҳбһ¶бһ“ confidence бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһҸбҹ’бһҗбһ»бһ“бһёбһҳбһҪбһҷбҹ—бҹ”

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ (Complete Pipeline)

бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (UNDER THE HOOD)

бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ YOLO (YOLO Architecture)

бһҳбҹүбһјбһҢбҹӮбһӣ YOLO бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹӮбһ„бһ…бҹӮбһҖбһҮбһ¶бһ”бһёбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһңбһҸбҹ’бһҗбһ»бһ”бһ¶бһ“бһӣбһҝбһ“ бһ“бһ·бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

01

Backbone (бһ•бҹ’бһ“бҹӮбһҖбһ‘бһ¶бһүбһҷбһҖбһӣбһҖбҹ’бһҒбһҺбҹҲ)

Backbone бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҸбһ¶бһү Convolutional Neural Network (CNN) бһҠбһјбһ…бһҮбһ¶ CSPDarknetбҹ” бһңбһ¶бһҳбһ¶бһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ–бһёбһҡбһјбһ”бһ—бһ¶бһ– бһҠбһјбһ…бһҮбһ¶бҹ–

  • бһӮбҹӮбһҳ (Edges)
  • бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ”бҹ’бһӣбһ„бҹӢ (Shapes & Structures)
  • бһңбһ¶бһҷбһ“бһ—бһ¶бһ– (Textures)
  • бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢбһңбһҸбҹ’бһҗбһ»
02

бһҡбһ¶бһӣбҹӢ Bounding Box бһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһӣбҹҒбһҒбһҹбҹҶбһҒбһ¶бһ“бҹӢ бҹӨ бһӮбҹҒбһ бҹ…бһҗбһ¶ x, y, w, h бһҠбҹӮбһӣбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ бҹҶбҹ”

Confidence Score бһӮбһәбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒ (0 бһҠбһӣбҹӢ 1) бһҠбҹӮбһӣ AI бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһңбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ вҖңбһ—бһ¶бһ–бһҮбһҝбһҮбһ¶бһҖбҹӢвҖқ бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһңбһҸбҹ’бһҗбһ»бһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһҸбһҳбҹ’бһӣбҹғбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҒбҹ’бһ–бһҹбҹӢ бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбһ¶бһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҡбһ”бһҹбҹӢбһңбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

бһҖбҹ’бһ“бһ»бһ„ Object Detection бһҠбһјбһ…бһҮбһ¶ YOLO бһҡбһ¶бһӣбҹӢ Bounding Box бһҳбһҪбһҷбһҳбһ·бһ“бһҳбҹӮбһ“бһҳбһ¶бһ“бһҸбҹӮбһ‘бһёбһҸбһ¶бҹҶбһ„бһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҖбҹҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ вҖңбһ–бһ·бһ“бҹ’бһ‘бһ»бһ—бһ¶бһ–бһҮбһҝбһҮбһ¶бһҖбҹӢвҖқ бһ•бһ„бһҠбҹӮбһҡбҹ” бһ–бһ·бһ“бҹ’бһ‘бһ»бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ вҖңбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ¬бһўбһҸбҹӢвҖқ бһ“бһ·бһ„ вҖңбһңбһҸбҹ’бһҗбһ»бһ“бҹ„бҹҮбһҮбһ¶бһўбҹ’бһңбһёвҖқбҹ”

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҳбҹүбһјбһҠбҹӮбһӣ Neural Network бһҠбҹӮбһӣбһҡбҹҖбһ“бһ–бһё data бһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһ–бҹҒбһӣбһңбһ¶бһҳбһҫбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҗбҹ’бһҳбһё бһңбһ¶бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһӣбһҖбҹ’бһҒбһҺбҹҲ (features) бһҠбҹӮбһӣбһңбһ¶бһ’бҹ’бһӣбһ¶бһ”бҹӢбһҡбҹҖбһ“ бһ бһҫбһҷбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһҗбһ¶ бһҸбһҫбһңбһҸбҹ’бһҗбһ»бһ“бҹ„бҹҮбһҹбҹ’бһҡбһҠбҹҖбһ„бһ“бһ№бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһңбһ¶бһ’бҹ’бһӣбһ¶бһ”бҹӢбһғбһҫбһүбҹ”

  • 1. Objectness Score (бһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ¬бһўбһҸбҹӢ)
    бһңбһ¶бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ бһҖбҹ’бһ“бһ»бһ„ Bounding Box бһҳбһҪбһҷ бһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһ¬бһўбһҸбҹӢбҹ” бһ”бһҫ score бһҒбҹ’бһ–бһҹбҹӢ вҶ’ бһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ”бһҫ score бһ‘бһ¶бһ” вҶ’ бһўбһ¶бһ…бһҮбһ¶бһ•бҹ’бһ‘бҹғбһ‘бһ‘бҹҒ бһ¬ noise
  • 2. Class Probability (бһ”бҹ’бһҡбһ—бҹҒбһ‘бһңбһҸбҹ’бһҗбһ»)
    бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбһ№бһ„бһҗбһ¶бһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ бһҫбһҷ бһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҗбһ¶бһңбһ¶бһҮбһ¶бһўбҹ’бһңбһё бһҠбһјбһ…бһҮбһ¶ person, car, dog, bicycle... бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһү вҖңбһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣвҖқ бһ“бҹғбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бһёбһҳбһҪбһҷбҹ—бҹ”
  • 3. Final Confidence Score
    бһӮбһәбһҖбһ¶бһҡбһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣ Objectness Г— Class Probability бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһ·бһ“бҹ’бһ‘бһ»бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Bounding Box бһҳбһҪбһҷбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣ YOLO бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһү бһ¬бһӣбһ»бһ”бһ…бҹ„бһӣбҹ”
  • 4. Confidence Threshold (бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһҸбҹӢбһ…бҹ„бһӣ)
    бһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһ”бҹ’бһҡбһўбһ”бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“ confidence бһ‘бһ¶бһ”бһҮбһ¶бһ„ threshold бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһҠбһ·бһҹбҹҒбһ’бҹ” бһңбһ¶бһҮбһҪбһҷбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ false detection бһ“бһ·бһ„ noiseбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ (real-world application) бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ threshold бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһ”бһҫбһҖбҹҶбһҺбһҸбҹӢбһҒбҹ’бһ–бһҹбҹӢбһ–бҹҒбһҖ вҶ’ бһўбһ¶бһ…бһ”бһ¶бһҸбҹӢ detection бһҒбҹ’бһӣбҹҮ бһ”бһҫбһ‘бһ¶бһ”бһ–бҹҒбһҖ вҶ’ бһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҒбһ»бһҹбһ…бҹ’бһҡбһҫбһ“бҹ”

4пёҸвғЈ бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„ бҹЎ Epoch

  1. Shuffle Data

    бһӣбһ¶бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ»бҹҶбһІбҹ’бһҷ model бһ…бһ„бһ…бһ¶бҹҶбһӣбҹҶбһҠбһ¶бһ”бҹӢ

  2. Forward Pass

    бһ”бһүбҹ’бһ…бһјбһӣбһҡбһјбһ” вҶ’ model бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷ bounding boxes + classes

  3. Compute Loss
    • Box Loss (CIoU) вҶ’ бһҖбҹҶбһ бһ»бһҹбһ‘бһёбһҸбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ бҹҶбһ”бҹ’бһҡбһўбһ”бҹӢ
    • Class Loss (BCE) вҶ’ бһҖбҹҶбһ бһ»бһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘ Class
    • DFL Loss вҶ’ бһҖбҹҶбһ бһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһўбһ”бҹӢ
  4. Backward Pass (Backpropagation)

    бһӮбһҺбһ“бһ¶ gradient бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Chain Rule бһ–бһё Calculus

  5. Optimizer Step

    бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң weights бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ SGD/Adam

  6. Validation

    бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбһҫ validation set (бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ epoch)

рҹ“җ

5пёҸвғЈ Loss Functions бһӣбһҳбҹ’бһўбһ·бһҸ

Box Loss (CIoU)
CIoU = 1 - IoU + (ПҒВІ(b,b_gt)/cВІ) + Оұv
IoU = Intersection over Union
ПҒ = distance between centers
Оұv = aspect ratio penalty
Class Loss (BCE)
BCE = -[yВ·log(Е·) + (1-y)В·log(1-Е·)]
бһңбһ¶бһҹбҹӢбһҖбҹҶбһ бһ»бһҹбһҡбһңбһ¶бһ„ Class бһ–бһ·бһҸ бһ“бһ·бһ„бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷ
DFL Loss
DFL = -ОЈ(y_i В· log(Е·_i))
бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ”бҹ’бһҡбһўбһ”бҹӢбһҠбҹӮбһӣбһ–бһ·бһ”бһ¶бһҖбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷ (hard examples)
Total Loss
Loss = ОұВ·BoxLoss + ОІВ·ClsLoss + ОіВ·DFLLoss
Оұ=7.5, ОІ=0.5, Оі=1.5 (default weights)

рҹҺҜ

Precision (P)

бһҡбһјбһ”бһҳбһ“бҹ’бһҸ
P = TP / (TP + FP)

TP = True Positive (бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң)
FP = False Positive (бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҒбһ»бһҹ)

вҶ’ бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

рҹ”Қ

Recall (R)

бһҡбһјбһ”бһҳбһ“бҹ’бһҸ
R = TP / (TP + FN)

TP = True Positive
FN = False Negative (бһҒбһҖбһҒбһ¶бһ“бһҳбһ·бһ“бһғбһҫбһү)

вҶ’ бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһғбһҫбһүбһңбһҸбҹ’бһҗбһ»бһ–бһ·бһҸбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

Pro Tips
  1. Start Small

    бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһҸбһјбһ… (nano/small) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ“бҹ’бһҹбҹҶбһ–бҹҒбһӣ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“

  2. Transfer Learning

    бһ”бҹ’бһҡбһҫ Pretrained Weights бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһӣбһҝбһ“ бһ‘бҹ„бҹҮбһ”бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ·бһ…бһҖбҹҸбһҠбҹ„бһҷ

  3. Balanced Dataset

    бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ…бҹҶбһ“бһҪбһ“бһҡбһјбһ”бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„ Class бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–

  4. Use Google Colab

    бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“ GPU бһ”бҹ’бһҡбһҫ Google Colab (бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ) бһҮбһ¶бһҳбһҪбһҷ GPU T4

  2. Test Different Thresholds

    бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ confidence (0.1-0.9) бһ“бһ·бһ„ iou (0.3-0.7) бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ
