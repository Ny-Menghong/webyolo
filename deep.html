🚀 You Only Look Once · Real-time Object Detection
🤔

តើ YOLO ជាអ្វី?

ស្វែងយល់

YOLO (You Only Look Once) គឺជា ក្បួនដោះស្រាយបញ្ញាសិប្បនិម្មិត មួយប្រភេទ ដែលប្រើប្រាស់ បណ្តាញសរសៃប្រសាទជ្រៅ (Deep Neural Network) ដើម្បី ស្វែងរក និង ចាត់ថ្នាក់ វត្ថុក្នុងរូបភាព ឬវីដេអូក្នុងពេលជាក់ស្តែង (Real-time)។

⚡

ហេតុអ្វី "You Only Look Once"?

  • ម៉ូដែលចាស់ៗ (R-CNN) ត្រូវ មើលរូបភាពច្រើនដង → យឺត
  • YOLO មើលរូបភាពទាំងមូលតែម្តង ហើយទស្សន៍ទាយប្រអប់ និង Class ក្នុងពេលតែមួយ
  • លទ្ធផល៖ លឿនណាស់ (៤៥–១៥៥ FPS)
🎯

គោលការណ៍ស្នូល

  • បំបែករូបភាពជា ក្រឡាចត្រង្គ (Grid) ឧ. ៧×៧
  • ក្រឡានីមួយៗទស្សន៍ទាយ ប្រអប់ B និង Class C
  • ប្រើ Non-Maximum Suppression (NMS) ដើម្បីដកប្រអប់ដែលត្រួតគ្នា

គណិតវិទ្យាខាងក្រោយ

Formula // ពិន្ទុទំនុកចិត្ត
Confidence = P(Object) × IoU(truth, pred)

// Loss Function សរុប
Loss = BoxLoss (CIoU) + ClsLoss (BCE) + DFLLoss
CIoU = Complete IoU
វាស់ចម្ងាយ និងសមាមាត្រ
BCE = Binary Cross-Entropy
វាស់កំហុស Class
DFL = Distribution Focal Loss
ផ្តោតលើប្រអប់ពិបាក
🏛️

ស្ថាបត្យកម្ម YOLO (៣ ផ្នែក)

YOLO មានរចនាសម្ព័ន្ធ ៣ កម្រិតធំៗ៖

🦴

1. Backbone

CSPDarknet
  • ទាញយកលក្ខណៈពិសេស (Features)
  • ប្រើ Convolution Layers ច្រើនស្រទាប់
  • បង្រួមរូបភាព បង្កើន Channels
  • ប្រើ Batch Normalization + SiLU
🔗

2. Neck

FPN+PAN
  • បញ្ចូលលក្ខណៈពីកម្រិតផ្សេងៗ
  • FPN (Feature Pyramid) – ពីតូច→ធំ
  • PAN (Path Aggregation) – ពីរាក់→ជ្រៅ
  • ជួយរកវត្ថុ តូច និងធំ បានល្អ
🎯

3. Head

Detection
  • ទស្សន៍ទាយ លទ្ធផលចុងក្រោយ
  • ប្រអប់ (x, y, w, h)
  • Objectness Score (ទំនុកចិត្ត)
  • Class Probabilities (ប្រភេទ)
📊

កំណែ YOLO ប្រៀបធៀប

កំណែ លក្ខណៈពិសេស ល្បឿន ភាពត្រឹមត្រូវ
YOLOv5 ពេញនិយម, ងាយស្រួល ⚡ លឿន 📈 មធ្យម
YOLOv8 ⭐ ទំនើប, គាំទ្រច្រើន ⚡⚡ លឿន 📈📈 ខ្ពស់
YOLOv9 PGI + GELAN ⚡⚡⚡ លឿនណាស់ 📈📈📈 ខ្ពស់ណាស់
YOLO-NAS Neural Architecture Search ⚡⚡⚡ លឿនណាស់ 📈📈📈 ខ្ពស់

⭐ សម្រាប់អ្នកចាប់ផ្តើម សូមប្រើ YOLOv8 ព្រោះងាយស្រួល និងមានឯកសារច្រើន។

🔬

របៀបដំណើរការ YOLO (ជំហាន)

  1. បំបែករូបភាពជា Grid S×S (ឧ. 13×13)
  2. ក្រឡានីមួយៗ ទស្សន៍ទាយប្រអប់ B (ឧ. 3) ដែលមាន (x, y, w, h, confidence)
  3. ក្រឡានីមួយៗ ទស្សន៍ទាយប្រូបាប៊ីលីតេ Class C
  4. គុណ confidence × class probability បានពិន្ទុចុងក្រោយ
  5. NMS (Non-Maximum Suppression) ដកប្រអប់ដែលត្រួតគ្នាចេញ

កូដហ្វឹកហាត់

Python from ultralytics import YOLO

# 1. ផ្ទុកម៉ូដែល Pretrained
model = YOLO("yolov8n.pt") # nano, small, medium, large, xlarge

# 2. ចាប់ផ្តើមហ្វឹកហាត់
results = model.train(
    data="dataset/dataset.yaml",
    epochs=150, # ចំនួនជុំ
    batch=16, # ទំហំបាច់
    imgsz=640, # ទំហំរូបភាពបញ្ចូល
    device=0, # 0=GPU, "cpu"=CPU
    lr0=0.01, # Learning Rate ដំបូង
    momentum=0.937, # សម្រាប់ SGD
    weight_decay=0.0005, # កាត់បន្ថយ Overfitting
    fliplr=0.5, # Data Augmentation
    mosaic=1.0, # Mosaic augmentation
    project="runs/train",
    name="my_yolo_model",
    plots=True
)
🔄

អ្វីកើតឡើងក្នុង ១ Epoch

  1. Shuffle Data – លាយលំដាប់រូបភាព
  2. Forward Pass – បញ្ចូលរូបភាព → បានទស្សន៍ទាយ
  3. Calculate Loss – គណនាកំហុសពី ៣ ផ្នែក៖
    • Box Loss (CIoU) – កំហុសប្រអប់
    • Cls Loss (BCE) – កំហុស Class
    • DFL Loss – ផ្តោតលើប្រអប់ពិបាក
  4. Backward Pass – គណនា Gradient ដោយប្រើ Chain Rule
  5. Optimizer Step – កែទម្ងន់ដោយប្រើ SGD/Adam
  6. Validate – សាកល្បងលើ Validation Set
📊

ប៉ារ៉ាម៉ែត្រសំខាន់ៗ

Epochs
ចំនួនដងដែលម៉ូដែលឃើញទិន្នន័យទាំងអស់។ ច្រើនពេក → Overfitting
Batch Size
ចំនួនរូបភាពក្នុងមួយជំហាន។ ធំ → លឿន ប៉ុន្តែត្រូវការ VRAM
Learning Rate
ទំហំជំហានកែទម្ងន់។ តូច → យឺត, ធំ → មិនចូលចំណុចល្អ
Data Augmentation
បច្ចេកទេសបង្វិល, ពង្រីក, ប្តូរពណ៌ ដើម្បីកាត់បន្ថយ Overfitting

Metrics សម្រាប់វាយតម្លៃ

កំឡុងពេលហ្វឹកហាត់ អ្នកនឹងឃើញតម្លៃទាំងនេះ៖

🎯

Precision (P)

រូបមន្ត
P = TP / (TP + FP)

TP = True Positive (ទស្សន៍ទាយត្រឹមត្រូវ)
FP = False Positive (ទស្សន៍ទាយខុស)

→ ចំនួនទស្សន៍ទាយត្រឹមត្រូវក្នុងចំណោមទស្សន៍ទាយទាំងអស់

🔍

Recall (R)

រូបមន្ត
R = TP / (TP + FN)

TP = True Positive
FN = False Negative (ខកខានមិនឃើញ)

→ ចំនួនទស្សន៍ទាយឃើញវត្ថុពិតទាំងអស់

📊

mAP50

Mean Average Precision @ IoU=0.5
mAP50 = 0.587

ស្តង់ដារវាស់ភាពត្រឹមត្រូវ នៅ IoU threshold 0.5

📊

mAP50-95

mAP ជាមធ្យមលើ IoU 0.5–0.95
mAP50-95 = 0.345

កាន់តែតឹងតែង – វាស់លើ thresholds ច្រើនកម្រិត

ការសាកល្បង (Testing / Inference)

បន្ទាប់ពីហ្វឹកហាត់រួច យើងអាចសាកល្បងម៉ូដែលលើរូបភាពថ្មី។

📝

កូដសម្រាប់វាយតម្លៃ

Python from ultralytics import YOLO

# ផ្ទុកម៉ូដែលដែលបានហ្វឹកហាត់
model = YOLO("runs/train/my_yolo_model/weights/best.pt")

# វាយតម្លៃលើ Validation Set
model.val(
    data="dataset/dataset.yaml",
    batch=16,
    imgsz=640,
    conf=0.25, # Confidence threshold
    iou=0.45 # IoU threshold សម្រាប់ NMS
)
🖼️

សាកល្បងលើរូបភាពថ្មី

Python # សាកល្បងលើរូបភាពមួយ
results = model("test.jpg", conf=0.25, iou=0.45)

# បង្ហាញលទ្ធផល
results[0].show() # បង្ហាញរូបភាព
results[0].save("output.jpg") # រក្សាទុក

# បោះពុម្ពព័ត៌មាន
for box in results[0].boxes:
    print(f"Class: {model.names[int(box.cls)]}")
    print(f"Confidence: {box.conf}")
    print(f"Coordinates: {box.xyxy}") # x1,y1,x2,y2

គន្លឹះសម្រាប់អ្នកចាប់ផ្តើម

Pro Tips
  1. ចាប់ផ្តើមពី ម៉ូដែលតូច (nano/small) ដើម្បីសន្សំពេល និងធនធាន
  2. ប្រើ Pretrained Weights (Transfer Learning) ដើម្បីរៀនលឿន
  3. ត្រូវប្រាកដថា ចំនួនរូបភាពក្នុង Class នីមួយៗមានតុល្យភាព
  4. បើគ្មាន GPU ប្រើ Google Colab (ឥតគិតថ្លៃ)
  5. តាមដាន mAP ជានិច្ច ដើម្បីដឹងថាម៉ូដែលកំពុងរៀនបានល្អឬអត់
