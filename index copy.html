What is Artificial Intelligence?

Artificial Intelligence is the field of building systems that perform tasks which normally require human intelligence — perceiving, reasoning, learning, and deciding.

A Brief History

  • 1950 — Alan Turing proposes the Turing Test.
  • 1956 — The term "Artificial Intelligence" is coined at Dartmouth.
  • 1980s — Expert systems rise, then an "AI winter" follows.
  • 1997 — Deep Blue defeats world chess champion Garry Kasparov.
  • 2012 — AlexNet ignites the deep learning boom.
  • 2015 — YOLO reframes detection as one regression problem.
  • 2020s — Large models and real-time vision go mainstream.

LEARNING FROM DATA

Machine Learning

Machine Learning gives computers the ability to improve at a task by learning from examples, instead of following hand-written rules.

Supervised Learning

Learns from labeled examples — inputs paired with correct answers — to predict labels for new data.

e.g. spam detection, YOLO training

Unsupervised Learning

Finds hidden structure in unlabeled data by grouping similar examples together.

e.g. customer segmentation

Reinforcement Learning

An agent learns by trial and error, receiving rewards or penalties for its actions.

e.g. game-playing agents, robotics
NEURAL NETWORKS

Deep Learning

Deep Learning uses layered neural networks to automatically learn features directly from raw data like pixels.

Neural Networks

Layers of connected nodes ("neurons") that transform input signals into predictions.

Hidden Layers

Intermediate layers between input and output that build increasingly abstract features.

Weights & Bias

Weights scale each input's importance; bias shifts the result — both are learned during training.

Activation Functions

Non-linear functions (like ReLU) that let networks model complex relationships.

THE CORE OF THIS COURSE

YOLO (You Only Look Once)

YOLO looks at an image exactly once and predicts every object's location and class in a single pass — that's what makes it fast enough for real time.

Why Is YOLO Fast?

Older detectors scanned an image in multiple stages — propose regions, then classify each one. YOLO treats detection as a single regression problem, predicting all boxes and classes in one forward pass through the network.

One-Stage Detector

The image is divided into a grid; each grid cell directly predicts bounding boxes and class probabilities — no separate proposal stage needed.

Bounding Box & Confidence

Each box is defined by position and size, plus a confidence score describing how certain the model is that an object exists there.

Class Prediction & NMS

Each box also gets class probabilities. Non-Maximum Suppression then removes duplicate, overlapping boxes, keeping only the best one per object.

The Complete Pipeline

UNDER THE HOOD

YOLO Architecture

Every YOLO model is built from three cooperating parts.

01

Backbone

A convolutional network (like CSPDarknet) that extracts raw visual features — edges, textures, shapes — from the input image.

02

Neck

Layers (like FPN/PANet) that merge features from different scales, so both small and large objects can be detected.

03

Head

The final layers that output bounding box coordinates, confidence scores, and class probabilities per grid cell.

HOW SURE IS THE MODEL?

Confidence Score

Each predicted box carries a confidence score — a mix of objectness (is there anything here?) and class probability (what is it?).

  • Objectness — likelihood a box actually contains an object.
  • Probability — likelihood of the predicted class being correct.
  • Threshold — boxes scoring below it are discarded. Try dragging the slider to filter the weaker detections on the left.
